Se define una variable llamada URL_InfoAutos que contiene una URL (dirección web) que apunta a un archivo JSON en línea.

Se utiliza la función fetch para hacer una solicitud a la URL. Esta función devuelve una promesa que se resuelve cuando se obtiene la respuesta del servidor.

El primer .then convierte la respuesta en formato JSON.

Luego, otro .then llama a la función MostrarData y pasa los datos JSON obtenidos como argumento.

La función MostrarData recibe un arreglo de objetos y recorre cada objeto en un bucle.

Para cada objeto en el arreglo, se toman ciertos valores (como imagen, nombre, moneda, costo, descripción y cantidad vendida) y se utilizan para construir un fragmento de HTML.
